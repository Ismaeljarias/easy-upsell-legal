# Easy Upsell — Legal Site

Static website for [Easy Upsell](https://apps.shopify.com), a Shopify app that shows post add-to-cart upsell offers with optional discounts.

Deployed via GitHub Pages.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page with app overview |
| `privacy.html` | Privacy Policy |
| `terms.html` | Terms of Service |
| `support.html` | Support & contact info |
| `styles.css` | Shared stylesheet |

## Deploy

This site is served via GitHub Pages from the `main` branch root.

To enable: **Settings → Pages → Branch: `main` / Folder: `/ (root)`**

## Local preview

No build step required. Open any `.html` file directly in your browser.
